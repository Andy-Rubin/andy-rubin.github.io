HTML

1.基本元素
HTML文档是一份结构化的文档，根元素总是<html…/>元素，该元素内通常包含<head…/>和<body…/>两个子元素（HTML5允许忽略它们，会隐式添加）这三个元素定义了HTML文档 的基本结构: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>基本元素</title>
</head>
<body>
    <!-- 采用标题1到6来输出文本 -->
    <h1>讲义</h1>
    <h2>记忆</h2>
    <h3>大量</h3>
    <h4>dalaing</h4>
    <h5>weuitn00</h5>
    <h6>safihhri</h6>
    <!-- 输出一条水平线-->
    <hr />
    <!-- 使用三个span文本-->
    <span>Tomcat</span>
    <span>Jerry</span>
    <span>Tom</span>
    <!-- 输出换行-->
    <br />
    <!-- 使用三个div定义三节 -->
    <div>Tomcat</div>
    <div>Tomcat</div>
    <div>Tomcat</div>
    <!-- 用p定义段落 -->
    <p>Tomcat</p>
    <p>Tomcat</p>
    <p>Tomcat</p>
</body>
</html>

文本格式相关元素 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>文本格式化元素</title>
</head>
<body>
    <span><b>加粗字体</b></span><br />
    <span><i>斜体字体</i></span><br />
    <span><b><i>加粗斜字体</i></b></span><br />
    <span><em>强调的文本</em></span><br />
    <p><strong>加粗文本</strong></p>
    <small><span>小号字体文本</span></small>
    <div>普通文本<sup>上标文本</sup></div>
    <span>普通文本<sub>下标文本</sub></span><br />
<!-- 指定文本从左向右（正常情况） 排列 -->
    <bdo dir="ltr">从左向右排列的文本</bdo><BR/>
<!-- 指定文本从右向左排列 -->
    <BDO dir="rtl">从右向左排列的文本</BDO><BR />
</body>
</html>
