Realtime Multi-Person 2D Pose Estimation using Part Affinity Fields

1.介绍
人体二维姿态估计——定位解剖关键点或“部分”的问题——主要集中在寻找个体的身体部位 [8,4,3,21,33,13,25,31,6,24]。在图像中推断多人的姿势，特别是社会参与的个人，提出了一套独特的挑战。首先，每个图像可能包含一个未知的人数，可以出现在任何位置或规模。第二，人与人之间的交互作用会产生复杂的空间干扰，由于接触、遮挡和肢体关节，使得部件间的关联变得困难。第三，运行时复杂性会随着映像中人员的数量增加而增加，这使得实时性能成为一个挑战。
一种常用的方法是使用一个人检测器，对每个检测器进行单人位姿估计。
这些自顶向下的方法直接利用了现有的技术进行单人姿势估计，但是早期的承诺是有问题的:如果人员检测器失败了——当人们接近时很容易出现这种情况——就没有恢复的办法了。此外，这些自顶向下方法的运行时间与人员数量成正比:对于每个检测，运行一个单人姿态估计器，人员越多，计算成本越大。相反，自底向上的方法很有吸引力，因为它们提供了对早期承诺的健壮性，并且有潜力将运行时复杂性与映像中的人员数量解耦。然而，自底向上的方法并不直接使用来自其他身体部位和其他人的全局上下文线索。在实践中，由于最后的解析需要昂贵的全局推理，以前的自底向上方法并没有保留效率的提高。例如，Pishchulin等人的开创性工作。提出了一种自底向上的方法，该方法联合标记部件检测候选对象并将它们与个人关联起来。然而，在全连通图上求解整数线性规划问题是一个np困难的问题，平均处理时间约为小时。Insafutdinov等人构建了基于ResNet和图像相关的成对分数的更强的部件检测器，并极大地改进了运行时，但该方法仍然需要几分钟来处理每张图像，并且限制了部件建议的数量。使用的成对表示很难精确地回归，因此需要单独的逻辑回归。
在这篇论文中，我们提出了一种有效的方法，可以在多个公共基准上获得最精确的多姿态估计。我们提出了第一个自底向上表示的关联分数通过部分亲和域(PAFs)，一组二维的vetor域编码的位置和方向的界限在图像域。我们证明，同时推断这些自底向上表示的检测和assosiation编码全局的conquality结果，只需要一小部分的计算成本。我们已经公开发布了代码的完全重现性，提出了第一个实时系统的多个人2D位姿检测。
2.方法
图片说明了我们方法的总体流程。系统以大小w X h的彩色图像作为输入，并输出图像中每个人的二维解剖关键点位置。首先，前馈网络同时预测一组零件位置的二维置信度图S和一组零件位置的二维向量场L和一组零件的二维向量场L，对零件之间的关联度进行编码。集合L有C个向量场，每个翼一个，其中Lc, Lc中的每个图像位置编码一个2D向量(如图1所示)。最后，通过贪婪推理解析置信度图和亲和域，输出图像中所有人的二维关键点。
